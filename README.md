# のれん診断 - 飲食店M&A売却価格診断

飲食店オーナー向けの無料M&A売却価格診断ツールです。

## 機能

- **簡易診断**: 7つの選択式質問で約1分で完了
- **詳細診断**: 財務情報入力で3つの評価手法による精密診断

### 3つの評価手法

1. **DCF法** - 将来キャッシュフローを現在価値に割引
2. **時価純資産法** - 資産価値 + 営業権（のれん代）
3. **マルチプル法** - 業界のEBITDA倍率を適用

## 技術スタック

- React 18
- Vite 5
- Tailwind CSS 3

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## デプロイ

Vercelへのデプロイを推奨します。

1. GitHubにリポジトリを作成
2. Vercelでプロジェクトをインポート
3. 自動でビルド・デプロイされます
